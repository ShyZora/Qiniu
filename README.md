需要启动redis服务且运行环境要有mysql

mysql用户名： root

mysql密码：admin

进入jar目录，依次执行
``` java
java -jar qiniu-category.jar
java -jar qiniu-video.jar
java -jar qiuniu-security.jar
```

即可启动服务
